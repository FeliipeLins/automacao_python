# 📊 Automação de Análise de Dados com Python

Projeto desenvolvido durante o curso da Hashtag Treinamentos com o objetivo de automatizar processos de análise de dados e envio de relatórios utilizando Python.

## 🚀 Tecnologias Utilizadas

* Python
* Pandas
* PyAutoGUI
* Pyperclip
* Time
* VS Code + Extensão Jupyter

## ⚙️ Funcionalidades da Automação

A automação realiza um fluxo completo de tarefas automaticamente:

* Abre o navegador
* Navega pela base de dados
* Realiza a análise das informações
* Gera um relatório com os dados analisados
* Envia o relatório automaticamente por e-mail

## 📚 Objetivos do Projeto

Este projeto foi desenvolvido para praticar:

* Automação de tarefas repetitivas
* Manipulação e análise de dados com Pandas
* Integração entre bibliotecas Python
* Otimização de processos
* Desenvolvimento de soluções automatizadas

## 💻 Ambiente de Desenvolvimento

O projeto foi desenvolvido no Visual Studio Code utilizando a extensão Jupyter, proporcionando uma experiência mais dinâmica para execução e análise do código.

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/FeliipeLins/automacao_python.git
```

2. Instale as dependências:

```bash
pip install pandas pyautogui pyperclip
```

3. Execute o projeto no VS Code utilizando a extensão Jupyter ou rode o script Python normalmente.

## 📈 Resultado

Com essa automação, foi possível reduzir tarefas manuais, aumentar a produtividade e automatizar todo o processo de análise e envio de informações.
